# Hamburger_Siparisi_Web_Anasayfasi — Backend

## Kurulum
```bash
pip install -r requirements.txt
```

## Çalıştırma
```bash
python backend_api.py
```

Backend **http://localhost:8000** adresinde çalışır.
API dökümantasyonu: **http://localhost:8000/docs**

## Teknolojiler
- Python
- FastAPI
- SQLAlchemy (SQLite)
